# REACT-MAC

This repository provides the Google Colab implementation of REACT-MAC, a lightweight MAC-layer protocol for detecting and mitigating selfish backoff behavior in IEEE 802.11 Industrial IoT networks.

## How to Run
Open the notebook in Google Colab and execute all cells sequentially.

## Requirements
Python 3.x, NumPy, Matplotlib (pre-installed in Colab).
